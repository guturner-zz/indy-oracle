Indy Oracle is a hub for cosplayers to register as volunteers on an escort team. If guests at a convention feel unsafe for any reason, they are encouraged to text the number above and an escort will get back to them ASAP.

How Does Indy Oracle Work?

Admins can invite users to join the website. If they're available to serve as an escort at a convention, users can opt into the program by providing their phone number. When guests text the number above, a copy of that text is sent to a secure inbox that is constantly being monitored by the Indy Oracle script. When Indy Oracle discovers an e-mail, it will text the details to those users that have opted in.
